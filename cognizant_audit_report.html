Data Minimisation Filter
A pre-processing pipeline strips PII from email evidence and redacts non-relevant fields from system logs before they enter the agent pipeline. Only control-relevant metadata (approver, timestamp, amount) is retained.
Evidence Chain of Custody
Every piece of evidence collected receives a tamper-evident hash at ingestion. The hash is stored in an immutable audit log. If an evidence artifact is modified after collection, the chain breaks and the finding is auto-flagged as compromised.
HITL & 4-Eye Escalation
All AI-generated control findings land in a "Pending Validation" queue. An Internal Auditor must mark each as Confirmed/False Positive. Any finding classified as "Material" triggers a mandatory 4-eye review by the Head of Internal Audit before it is escalated to PWC.
Vision Model Guardrail
The Vision/OCR agent operates within a sandboxed execution environment. It can only read screenshots from approved SharePoint folders. Output is limited to structured JSON (approver name, date, amount) — no freeform text generation permitted from visual inputs.

The AI produces a ranked scorecard with transparent confidence intervals. The Procurement Board (minimum 3 members) reviews the scorecard in a formal session. The award decision is documented with individual board member rationale — the AI score is advisory input, not determinative.
